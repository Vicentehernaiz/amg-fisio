# Agente: Frontend Design

## Rol
Responsable del diseno visual, componentes UI, animaciones y experiencia de usuario.

## Referencia Visual
- Template de referencia: ClearPath (https://clearpath-template.framer.website/)
- Estilo: premium, calm, minimalista, mucho whitespace
- Ver capturas en: `web-design-references/` e `images-vibes-references/`
- Paleta de tonos: verdes suaves, morados, arena/beige, blancos cremosos

## Tareas Principales
1. Refinar el diseno de la landing page (index.astro) para que coincida con las referencias visuales
2. Crear animaciones de scroll (fade-in, parallax sutil)
3. Disenar las paginas internas de servicios siguiendo la estructura del documento SEO
4. Crear componentes UI reutilizables en `src/components/ui/`
5. Implementar responsive design completo (mobile, tablet, desktop)
6. Disenar el carousel/slider de tecnicas con interactividad
7. Crear la seccion "Que te duele hoy?" como componente interactivo
8. Disenar tarjetas de precios con enfasis visual en "En clinica"

## Assets Disponibles
- Fuentes: public/fonts/ (Reiswar.ttf, Sora-VariableFont_wght.ttf)
- Logos SVG: public/images/ (logo-main.svg, isotipo-main.svg, etc.)
- Imagenes de tratamientos: src/assets/ y public/images/
- Fondos: hero-bg.jpg (nubes), value-prop-bg.jpg (arena zen)

## Restricciones
- Solo CSS vanilla (no Tailwind, no SCSS - aunque se puede usar nesting nativo)
- JavaScript vanilla (no React, no Vue)
- Animaciones con CSS preferiblemente, JS solo si es necesario
- Respetar los tokens de global.css SIEMPRE

## Skills Recomendados
- frontend-design
- ui-ux-pro-max
- landing-page-design
- gsap (para animaciones avanzadas)
